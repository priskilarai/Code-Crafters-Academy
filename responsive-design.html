Introduction to Responsive Design

Responsive web design ensures that websites work well on all devices and screen sizes. It combines flexible layouts, media queries, and scalable images to create optimal viewing experiences.

Viewport Meta Tag

The viewport meta tag controls how a webpage is displayed on mobile devices. It's essential for responsive design.

Essential Viewport Configuration

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<!-- This tag ensures proper rendering and touch zooming on mobile devices -->

Media Queries

Media queries allow you to apply different styles based on device characteristics like screen width, height, and orientation.

Interactive Media Query Demo

Desktop 1200px+

Resize your browser to see this element change!

Media Query Examples

/* Mobile First Approach */
.container {
    width: 100%;
    padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        max-width: 750px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        max-width: 1200px;
        padding: 2rem;
    }
}

/* Large Desktop */
@media (min-width: 1440px) {
    .container {
        max-width: 1400px;
    }
}

Responsive Images

Optimize images for different screen sizes and resolutions to improve performance and user experience.

Responsive Image Techniques

CSS-based Responsive Image

img {
    max-width: 100%;
    height: auto;
    display: block;
}

Picture Element with Multiple Sources

<picture>
    <source media="(min-width: 1024px)" srcset="large.jpg">
    <source media="(min-width: 768px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Description">
